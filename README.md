# Run Calculator

A basic calculator application built with PHP, HTML, and CSS.

## Features
- Basic arithmetic operations: add, subtract, multiply, divide  
- Simple and clean interface  
- Easy to run locally

## How to Run
1. Clone the repository  
   ```bash
   git clone https://github.com/Prasann62/run_calculater.git

Move into the folder
cd run_calculater
Place the folder inside your web server root (e.g., htdocs for XAMPP)

Open your browser to:

http://localhost/run_calculater/
